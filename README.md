blog.wangningbo.com
=========

[View Site](https://blog.wangningbo.com)

Building
--------

```bash
git clone git@github.com:hqwangningbo/hqwangningbo.github.io.git
cd hqwangningbo.github.io
npm install
npm run dev
# Open localhost:4003 in your browser
```

Building for Production
--------

```bash
npm run build
```
